Como desarrollador fullstack, tengo experiencia en el desarrollo con PHP y el framework Laravel, que utilizo para crear aplicaciones web robustas y eficientes. Combino estos conocimientos con JavaScript, CSS y HTML para diseñar interfaces atractivas e interactivas, siempre buscando optimizar la experiencia del usuario en cada proyecto.

También trabajo con bases de datos, lo que me permite construir sistemas sólidos y escalables, asegurando que la información se almacene de forma segura y eficaz.

Mi enfoque abarca tanto el desarrollo del backend con PHP y Laravel, como la creación de interfaces dinámicas en el frontend. Esto me permite ofrecer soluciones completas, desde la planificación inicial hasta el despliegue final.

Con un buen manejo de estas tecnologías y entendiendo cómo se relacionan entre sí, me preparo para enfrentar distintos desafíos y transformar ideas en proyectos web funcionales y visualmente atractivos.

Actualmente, estoy cursando la carrera de Analista Programador Universitario en la Universidad Nacional de La Plata (UNLP), desde 2024.
